# Pages

Main pages for TravelMate web app.

## Files
- **Home.tsx**: Landing page showing featured destinations and popular trips.
- **Destinations.tsx**: Search and list travel destinations, fetches weather data.
- **TravelJournal.tsx**: CRUD for travel journal entries, linked to user authentication.
- **Auth/**: Contains login and registration pages.

Each page is a React component and uses context/services for data.
